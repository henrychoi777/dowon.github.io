====================================
🚀 다음 번에 작업 시작하는 순서
====================================

**1단계: 폴더 이동**
cd c:\dsw

**2단계: 의존성 설치 (처음 1회 또는 package.json 변경 시)**
npm install

**3단계: 서버 시작** ⭐ 가장 중요!
node server.js

**4단계: 웹 접속**
VS Code 미리보기에서 http://localhost:3000 접속
또는 브라우저에서 http://localhost:3000 접속

====================================
✅ 체크리스트
====================================

서버 시작 후 확인사항:
☑ 메인 페이지 (/) 로드되는지 확인
☑ 등록서류함 (/documents) 클릭 테스트
☑ 서류제출하기 (/submissions) 클릭 테스트
☑ 우측 하단 관리자 (🔐) 버튼 클릭 가능
☑ 콘솔에 에러 없는지 확인

====================================
📝 주요 파일
====================================

- c:\dsw\server.js → Express 서버 메인 파일
- c:\dsw\.env → API 키 저장 (절대 GitHub에 푸시 금지!)
- c:\dsw\submissions.json → Supabase 마이그레이션 후 사용 안 함
- c:\dsw\supabase-schema.sql → Supabase 테이블 생성 SQL

====================================
🔐 Supabase 설정 (아직 안 했으면)
====================================

1. Supabase 콘솔 접속: https://app.supabase.com
2. SQL Editor에서 supabase-schema.sql 내용 실행
3. Storage에서 "submissions" bucket 생성 (공개)

====================================
⚠️ 문제 발생 시
====================================

서버가 안 켜질 때:
taskkill /F /IM node.exe

포트 충돌 (3000번 이미 사용 중):
netstat -ano | findstr :3000
taskkill /PID [PID번호] /F

변경된 코드 적용 안 될 때:
1. Ctrl+C로 서버 중지
2. node server.js 다시 실행
